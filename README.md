# OpenStreetMap Location Tracker

A Flutter-based **location** tracker app that shows user location on an OpenStreetMap-powered map and includes a simple profile page.

---

## Overview

This app allows users to view their current location on an interactive OpenStreetMap map and manage basic profile information in a dedicated profile screen.

Built with Flutter and OpenStreetMap map tiles, it is suitable as a starter template for location-aware apps such as delivery tracking, travel, or check-in apps.

---

## Features

- View current location on OpenStreetMap
- Pan and zoom the map interactively  
- Profile page with basic user info
- Simple, clean UI suitable for extension
- Android support
